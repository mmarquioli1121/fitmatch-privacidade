# Política de Privacidade — FitMatch

Última atualização: 21/01/2025

O FitMatch respeita a privacidade dos seus usuários e está comprometido em proteger todos os dados pessoais coletados. Esta Política de Privacidade descreve como coletamos, utilizamos, armazenamos e protegemos suas informações ao utilizar nosso aplicativo.

---

## 1. Dados coletados

O FitMatch coleta os seguintes dados para funcionamento do app:

### 1.1 Informações fornecidas pelo usuário
- Nome
- Foto de perfil
- Esportes praticados
- Nível de habilidade
- Biografia (opcional)
- Eventos criados ou participados
- Mensagens enviadas no chat dos eventos

### 1.2 Informações coletadas automaticamente
- Endereço de email usado no login
- Localização aproximada e/ou precisa (GPS)
- Identificadores de dispositivo
- Informações de uso do app

---

## 2. Uso das informações

As informações coletadas são utilizadas para:

- Criar e gerenciar sua conta
- Exibir eventos esportivos próximos
- Calcular distância entre usuários e locais dos eventos
- Permitir que você participe de eventos
